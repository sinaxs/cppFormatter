# cppFormatter
## Build

```sh
cd build
cmake ..
make
make install
```
## Usage

In your c code add this:

```c
#include <cppFormatter/cppFormatter.h>
```

List of Functions:

```c
string printRed(TYPE input);
string printYellow(TYPE input);
...
```
